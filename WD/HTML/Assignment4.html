Assignment4:HTML List Tags Examples

Unordered List

An unordered list displays items with bullet points:

  • Apple
  • Banana
  • Cherry

Ordered List

An ordered list displays items with numbers:

  1. Step 1: Prepare the ingredients
  2. Step 2: Cook the dish
  3. Step 3: Serve and enjoy

Nested Lists

You can nest lists inside each other:

  • Fruits
    • Apple
    • Banana
  • Vegetables
    • Carrot
    • Spinach

Description List

A description list is used to define terms with descriptions:

HTML
A markup language for creating web pages.
CSS
A stylesheet language for styling web pages.
JavaScript
A programming language for creating dynamic web content.

Ordered List with Types

Change the numbering style using the type attribute:

Type1:Numbers

  1. Coffee
  2. Tea
  3. Milk

Type2:Roman Numbers - Uppercase

  1. First Item
  2. Second Item
  3. Third Item

Type3:Roman Numbers - lowercase

  1. First Item
  2. Second Item
  3. Third Item

Type4:Uppercase Letters

  1. Option A
  2. Option B
  3. Option C

Type5:Lowercase Letters

  1. Option A
  2. Option B
  3. Option C
